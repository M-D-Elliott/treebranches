Controls:
  • Right-click folder or file for context menu
  • N - New Folder
  • F - New File
  • Arrow Keys - Next/Prev Folder/file
  • CTRL + X - Cut
  • CTRL + C - Copy
  • CTRL + Z - Undo
  • CTRL + SHIFT + Z (OR) CTRL + Y - Redo
  • Delete - Move to Trash / Delete from Trash / Empty Trash
  • Enter - Expand / Collapse Folder
  • S - Sort by LAST SORT
  • R - Rename
